# Tic Tac Toe

A modern, responsive Tic Tac Toe game built with vanilla HTML, CSS, and JavaScript.

![Tic Tac Toe](https://img.shields.io/badge/version-1.0-blue) ![License](https://img.shields.io/badge/license-MIT-green)

## Features

- **Modern Design** - Sleek dark gradient UI with neon accents
- **Responsive** - Works on desktop, tablet, and mobile devices
- **Score Tracking** - Keeps track of wins for X, O, and draws
- **Smooth Animations** - Pulsing animation for winning combinations
- **Instant Reset** - Start a new game at any time

## How to Play

1. Player X always goes first
2. Click any empty cell to place your mark
3. Get three in a row (horizontal, vertical, or diagonal) to win
4. The game tracks your score across multiple rounds
5. Click "New Game" to reset the board (scores persist)

## Getting Started

### Option 1: Python
```bash
python3 -m http.server 8000
```

### Option 2: Node.js
```bash
npx serve .
```

### Option 3: Just Open It
Simply open `index.html` directly in your browser.

Then visit http://localhost:8000 in your browser.
